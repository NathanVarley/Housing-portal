# Aegis Wellness Engine

A Flet-based wellness companion app designed to provide mental health support through grounding exercises, local resource discovery, and AI-powered chat assistance.

## Features

- **Grounding Exercise Component**: Interactive breathing/pulse animation exercise for mindfulness and relaxation
- **Local Support Resources**: Discover nearby wellness centers and support groups
- **AI Support Chat**: Real-time chat interface for mental health support conversations
- **Dark Theme UI**: Easy-on-the-eyes dark mode interface

## Requirements

- Python 3.8+
- Flet framework

## Installation

1. Clone this repository
2. Install dependencies:
```bash
pip install flet
```

## Running the App

```bash
python main.py
```

## Components

### Grounding Exercise
An animated pulse that expands and contracts to guide users through breathing exercises. Perfect for anxiety management and mindfulness.

### Resources Screen
Displays local wellness centers and support services with distances. Helps users find nearby help.

### Chat Interface
Simple messaging interface for connecting with AI support or counselors.
